1.strcpy和memcpy

strcpy和mempcy都是标准C库函数，它们有下面的特点。
strcpy提供了字符串的复制。即strcpy只用于字符串复制，并且它不仅复制字符串内容之外，还会复制字符串的结束符。
memcpy提供了一般内存的复制。即memcpy对于需要复制的内容没有限制，因此用户更广。
strcpy的函数原型：
char strcpy(char dest, const char src);
memcpy的函数运行
char memcpy(void dest, const void src, size_t count);

strcpy和memcpy的相同点：
都是可以实现字符串拷贝的功能
strcpy和memcpy的不同点：
（1）实现功能不同。strcpy只能复制字符串，mempcy可以进行内存块间的拷贝，即可以拷贝存在内存块中的任意内容，例如字符数组、整型、结构体、类等；
（2）复制的方法不同。strcpy不需要指定长度，它遇到被复制字符的串结束符”\0”才结束，所以容易溢出。memcpy则是根据其第3个参数决定复制的长度。
（3）执行效率不同。memcpy最高，strcpy次之。
